**Jak2Doom2**



A custom Doom mod for GZDoom featuring new weapons, enemies, and maps inspired by Jak 2



\---



Installation (Easy)



1\. Download and install **GZDoom**:  



https://zdoom.org/downloads



2\. Download this mod (`**Jak2Doom2.pk3**` file)



3\. Drag and drop the `**Jak2Doom2.pk3**` file onto `gzdoom.exe`


**The game will launch automatically with the mod**



                        

                      -- Important --





Requirements :


\- **GZDoom** (latest version)



\- **DOOM II** (doom2.wad required)



You must own a copy of **DOOM II** to **play** this mod.



The IWAD file (doom2.wad) is **not included.**



Alternative:



**Freedoom** can be used, but compatibility is not guaranteed.



The **Steam** version works perfectly
